# Cybersecurity Toolkit

## Overview

The Cybersecurity Toolkit is a collection of practical Python-based security tools designed to demonstrate fundamental cybersecurity concepts and security automation techniques.

This project showcases skills in password security, network scanning, and log analysis that are commonly used in IT support, network administration, and cybersecurity environments.

---

## Tools Included

### Password Strength Checker

Analyzes password complexity based on:

* Length
* Uppercase characters
* Lowercase characters
* Numbers
* Special characters

The tool classifies passwords as:

* Weak
* Moderate
* Strong
